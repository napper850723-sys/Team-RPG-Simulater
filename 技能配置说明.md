# 技能配置说明

## 概述

现在所有技能都统一配置在 `skills.js` 文件中，只需要在一个地方添加新技能即可。

## 添加新技能的步骤

### 1. 在 skills.js 中添加技能配置

打开 `skills.js` 文件，在 `skillList` 数组中添加新技能（在注释 `// 添加新技能在这里...` 下方）：

```javascript
{
    id: 'thunderStrike',           // 技能唯一标识（英文）
    name: '雷霆一击',               // 技能名称（中文）
    description: '召唤雷电攻击敌人，造成1.5倍攻击伤害并降低敌人速度',  // 技能描述
    targetType: 'single',          // 目标类型：single(单体)/all(全体)/allyAll(己方全体)/enemyAll(敌方全体)/self(自身)
    damageMultiplier: 1.5,         // 伤害倍率（可选）
    effect: 'thunderStrike',       // 效果标识（用于game.js中的switch case）
    priority: 1,                   // 优先级（数字越大优先级越高）
    cooldown: 3,                   // 冷却回合数
    releaseType: 'active'          // 释放类型：active(主动)/aura(光环)/death(死亡触发)
}
```

### 2. 在 game.js 中添加技能效果处理

打开 `game.js` 文件，找到 `applySkillEffect` 方法，在 switch 语句中添加新效果：

```javascript
case 'thunderStrike': 
    target.speed = Math.floor(target.speed * 0.85);  // 降低15%速度
    if (showLog) this.log(`${target.name} 被雷电击中，速度降低了！`); 
    break;
```

### 3. 在角色模板中使用新技能

打开 `select.js` 文件，找到角色模板数组（`playerTemplates` 或 `enemyTemplates`），直接使用技能ID配置技能：

```javascript
{ name: "雷神", hp: 130, speed: 14, attack: 28, defense: 8, skills: ['fireball', 'thunderStrike'] }  // 火球术+雷霆一击
```

**注意**：技能ID就是在 `skills.js` 中定义的 `id` 字段，如 `'fireball'`、`'thunderStrike'` 等。

## 参数说明

| 参数 | 必填 | 说明 | 可选值 |
|------|------|------|--------|
| `id` | 是 | 技能唯一标识 | 英文字符串，如 `'fireball'` |
| `name` | 是 | 技能显示名称 | 中文字符串，如 `'火球术'` |
| `description` | 是 | 技能描述 | 字符串 |
| `targetType` | 是 | 目标类型 | `single`/`all`/`allyAll`/`enemyAll`/`self` |
| `damageMultiplier` | 否 | 伤害倍率 | 数字，如 `1.2` 表示1.2倍伤害 |
| `effect` | 否 | 效果标识 | 字符串，用于 `applySkillEffect` 中的 switch case |
| `effectValue` | 否 | 效果数值 | 如复活技能的恢复百分比 `0.8` |
| `priority` | 是 | 技能优先级 | 数字，越大优先级越高 |
| `cooldown` | 是 | 冷却回合 | 数字，`0` 表示无冷却 |
| `releaseType` | 是 | 释放类型 | `active`/`aura`/`death` |

## 释放类型说明

- **active**：主动技能，角色行动时自动释放
- **aura**：光环技能，角色存活时持续生效
- **death**：死亡触发技能，角色死亡时自动释放

## 示例：添加一个"雷霆一击"技能

### 1. 在 skills.js 中添加：

```javascript
{ 
    id: 'thunderStrike', 
    name: '雷霆一击', 
    description: '召唤雷电攻击敌人，造成1.5倍攻击伤害并降低敌人速度',
    targetType: 'single',
    damageMultiplier: 1.5,
    effect: 'thunderStrike',
    priority: 1,
    cooldown: 3,
    releaseType: 'active'
}
```

### 2. 在 game.js 的 applySkillEffect 中添加：

```javascript
case 'thunderStrike': 
    target.speed = Math.floor(target.speed * 0.85);
    if (showLog) this.log(`${target.name} 被雷电击中，速度降低了！`); 
    break;
```

### 3. 在 select.js 的角色模板中使用：

```javascript
{ name: "雷神", hp: 130, speed: 14, attack: 28, defense: 8, skills: ['fireball', 'thunderStrike'] }
```

完成！新技能已添加并可以使用了。
